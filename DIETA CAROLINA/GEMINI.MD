# PROJETO: DIETA CAROLINA 🏋️‍♀️🥗

## 📝 Visão Geral
Este projeto é uma aplicação web mobile-first desenvolvida exclusivamente para **Carolina Duque**. O objetivo é facilitar o acompanhamento de um plano nutricional personalizado, permitindo visualização rápida da rotina diária, metas de macronutrientes, hidratação e progresso físico.

## 🚀 Tecnologias Utilizadas
- **Frontend**: HTML5, CSS3 (Vanilla)
- **Lógica**: JavaScript (ES6+)
- **Ícones**: Lucide Icons
- **Estilo**: Design moderno com Glassmorphism, otimizado para dispositivos móveis (iOS/iPhone).

## 📱 Funcionalidades Principais
1.  **Timeline Diária Inteligente**: Exibição das refeições baseada no dia da semana (Dia Personal, Dia Sozinha, Cardio/Descanso).
2.  **Gestão de Macronutrientes**: Resumo diário de Calorias (Kcal), Proteínas (P), Carboidratos (C), Gorduras (G) e Fibras.
3.  **Controle de Hidratação**: Sistema interativo para registrar o consumo de água com barra de progresso.
4.  **Sistema de Variações**: Menu lateral (Bottom Sheet) que permite trocar refeições por opções equivalentes sem quebrar a dieta.
5.  **Acompanhamento de Bioimpedância**: Histórico e análise clínica das métricas corporais.
6.  **Guia de Suplementação**: Resumo dos suplementos e horários recomendados.

## 🔄 Fluxos de Treino
O aplicativo adapta o cardápio automaticamente para três cenários:
-   **Dia Personal (Ter/Qui)**: Foco em Treino MMII pesado às 16h, com pré-treino leve focado em energia sustentada (Palatinose).
-   **Dia Sozinha (Qua/Sex)**: Treino às 14h com pré-treino reforçado.
-   **Cardio/Descanso (Seg/Sab/Dom)**: Distribuição equilibrada de macros para dias de menor intensidade.
